# **App Name**: Bullionaire Bot

## Core Features:

- Daily Configuration: Allow users to set daily risk limits or profit targets for automated trading.
- AI-Powered Trading Engine: Automate gold trading using AI algorithms, adapting to market conditions and executing trades 24/7.
- Real-Time Dashboard: Provide a live dashboard displaying current balance, profit/loss, active trades, and performance statistics.
- Risk Management System: Implement safeguards such as daily loss limits, maximum position sizes, and news event filters to protect user capital.
- Tiered Subscription Plans: Offer various subscription tiers with different trading limits, features, and accuracy targets.
- Referral Program: Incentivize users to refer friends with discounts and accuracy boosts. Include referral dashboard.
- Tomorrow's Suggestion Tool: Provide an AI-driven recommendation for position sizes for the next day based on market volatility, after analyzing previous trading data and market conditions using technical indicators.

## Style Guidelines:

- Primary color: Deep gold (#D4AF37) to convey value, class, and sophistication.
- Background color: Dark gray (#333333) to create a professional and modern feel.
- Accent color: Teal (#008080) for interactive elements and key metrics, providing a contrasting pop.
- Headline font: 'Playfair', serif, for headlines and short amounts of text. Body text: 'PT Sans', sans-serif.
- Use minimalist, professional icons related to trading and finance.
- Design a clean and intuitive layout for the dashboard, emphasizing key metrics and easy navigation.
- Subtle animations for loading states and trade executions.
